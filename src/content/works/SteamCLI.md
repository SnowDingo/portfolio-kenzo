Steam CLI is a command line interface (CLI) application that allows you to see public steam information directly from terminal using Steam Web API.

Skills I've gained:

* How to publish and maintain a npm package
* How to use React Inc to build fully responsive CLI application
* How to use Node.js to access APIs securely
