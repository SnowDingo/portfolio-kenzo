Lucky Omikuji is a simple omikuji/fortune telling app that uses randomness to tell user's fortune for the day.

You can visit the finished website [here](https://luckyomikuji.pages.dev).

Skills I've gained:

* How to use localstorage to store local data and prevent users from taking certain actions
* How to use React Hooks to create effective minimalistic application
* How to use tailwind CSS v4 to create an elegant design


## Tech Stack
This project uses:

* React
* RWC
* Vite.js
